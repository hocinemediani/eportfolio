Personal projects :

Web Phone book with Flask

Ever wanted to be able to have your own digital phone book with a pretty web interface that is easy to understand and utilize ? Then this project is made for you. Using HTML5 / CSS and Python, this web page is able to store and restore entries the same way a regular phone book would do.

Evander's Adventures

Evander's Adventures is a simple 2D platformer game made with Unity using C#. Embodying Evander, an adventurer, you'll embark in a journey full of twists, in search of the truth in a pixelated world full of lies. Explore, slash your enemies and find your way through while enjoying the landscapes.

Huffman File Compression

Using Huffman's compression algorithm, this program is able to compress and decompress any binary file with ease and without losing a single piece of information. No more lack of storage space will ever get to you, as storing and then re-utilizing files will take less space than ever !

UART Nexys 4

UARTs have been always used for serie communication between a microcontroler and one of its serie ports. This project simulates a UART full receiving/emitting communication on the Nexys 4 DDR FPGA card, using a minicom terminal to communicate with the card.

Simple's Adventure

Simple is a pretty simple guy. And like any other simple guy, he strives for one thing : POWER! Play as Simple and explore an entirely unique world made of multiple regions with their own sets of ressources, challenges, and enemies. Slay your way through every obstacles on the path to glory and become more than just another simple guy.

Docker ISP simulation

ISPs have always been seen has granted but the way they function has always been a blackbox for me. To better understand this, this project focuses on recreating an Autonomous System that acts as an ISP, interconnecting other ASs and granting a private and safe access to the rest of the network for its customers.
